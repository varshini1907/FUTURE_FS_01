## ✨ Features

* **📱 Fully Responsive Design:** Fluid layouts optimized for flawless viewing across desktops, tablets, and mobile devices.
* **🌙 Modern Dark UI:** A visually striking dark theme built with high-contrast elements for premium readability.
* **🗂️ Multi-Page Architecture:** Dedicated pages to cleanly separate your bio, engineering projects, and credentials.
* **💻 Project Showcase:** Clean grid layouts featuring cards for individual projects, complete with links to live demos and repositories.
* **📜 Certificates Gallery:** A structured section to highlight your professional certifications and achievements.
* **✨ Animated Effects:** Subtle scroll-triggered animations and hover transitions that elevate the overall user experience without compromising performance.
